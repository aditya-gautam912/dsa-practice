# Search Insert Position

**LeetCode #35** | Difficulty: Easy

## Problem

Given a sorted array `nums` and a `target`, return the index if found. Otherwise, return the index where it would be inserted in order.

### Examples

**Example 1:**
```
Input: nums = [1,3,5,6], target = 5
Output: 2
```

**Example 2:**
```
Input: nums = [1,3,5,6], target = 2
Output: 1
```

**Example 3:**
```
Input: nums = [1,3,5,6], target = 7
Output: 4
```

### Constraints

- `1 <= nums.length <= 10^4`
- `-10^4 <= nums[i] <= 10^4`
- `nums` contains distinct values sorted ascending.
