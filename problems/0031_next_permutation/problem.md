# Next Permutation

**LeetCode #31** | Difficulty: Medium

## Problem

A permutation of an array of integers is an arrangement of its members into a sequence or linear order.

The next permutation of an array of integers is the next lexicographically greater permutation. If no such arrangement exists, rearrange into the lowest possible order (ascending).

Given an array `nums`, modify it in-place to the next permutation.

### Examples

**Example 1:**
```
Input: nums = [1,2,3]
Output: [1,3,2]
```

**Example 2:**
```
Input: nums = [3,2,1]
Output: [1,2,3]
```

**Example 3:**
```
Input: nums = [1,1,5]
Output: [1,5,1]
```

### Constraints

- `1 <= nums.length <= 100`
- `0 <= nums[i] <= 100`
