# Search in Rotated Sorted Array

**LeetCode #33** | Difficulty: Medium

## Problem

Given a rotated sorted array `nums` and a `target`, return the index of `target` if found, else `-1`.

The array was originally sorted in ascending order then rotated at an unknown pivot.

### Examples

**Example 1:**
```
Input: nums = [4,5,6,7,0,1,2], target = 0
Output: 4
```

**Example 2:**
```
Input: nums = [4,5,6,7,0,1,2], target = 3
Output: -1
```

**Example 3:**
```
Input: nums = [1], target = 0
Output: -1
```

### Constraints

- `1 <= nums.length <= 5000`
- All values are unique.
- `-10^4 <= nums[i] <= 10^4`
