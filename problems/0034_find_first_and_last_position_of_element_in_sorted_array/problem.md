# Find First and Last Position of Element in Sorted Array

**LeetCode #34** | Difficulty: Medium

## Problem

Given a sorted array `nums` and a `target`, find the starting and ending position of the target.

Return `[-1, -1]` if target not found.

### Examples

**Example 1:**
```
Input: nums = [5,7,7,8,8,10], target = 8
Output: [3,4]
```

**Example 2:**
```
Input: nums = [5,7,7,8,8,10], target = 6
Output: [-1,-1]
```

**Example 3:**
```
Input: nums = [], target = 0
Output: [-1,-1]
```

### Constraints

- `0 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`
